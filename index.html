Smallr

I noticed that it's difficult to find small pets for adoption rather than purchase, so I made Smallr. Smallr was designed to help shelters and rescues showcase their small pets up for adoption. It is a full CRUD web application that allows users to browse small pets up for adoption or list a pet for adoption. Shelters must create an account to be able to create posts, as well as edit and/or delete their posts. This project is hosted on Heroku.

Node.js, Express.js, MongoDB, Mongoose, Passport.js, Google OAuth.
